Installing edge-tts in a Virtual Environment

What is edge-tts?

Edge-tts is a popular, free Python library and command-line tool that allows users to access Microsoft Edge's online text-to-speech service directly. It enables high-quality, natural-sounding voice synthesis in many languages without requiring an API key, the Edge browser, or a Windows operating system

What is a Virtual Environment?

A virtual environment is an isolated Python installation on your system. It allows you to install packages for specific projects without affecting your system Python installation. This prevents conflicts and keeps your system clean.

Step-by-Step Instructions

  • Step 1: Create a Directory for Your Project

    Open a terminal and create a directory where you want your project to live:

    bash
    mkdir my-edge-tts-project cd my-edge-tts-project

    Replacemy-edge-tts-project with whatever you'd like to name your project.

  • Step 2: Create the Virtual Environment

    Create a virtual environment in the current directory:

    bash
    python3 -m venv venv

    This creates a folder called venv containing an isolated Python installation.

  • Step 3: Activate the Virtual Environment

    Activate the virtual environment:

    bash
    source venv/bin/activate

    After running this, your command prompt should change to show something like:

    (venv) linux_user@boxDemon:~/my-edge-tts-project$

    The(venv) prefix indicates the virtual environment is active.

  • Step 4: Upgrade pip (Optional but Recommended)

    Update pip to the latest version:

    bash
    pip install --upgrade pip
  • Step 5: Install edge-tts

    Now install edge-tts:

    bash
    pip install edge-tts

    This should complete without errors since you're installing into the virtual environment, not system-wide.

  • Step 6: Verify Installation

    Test the installation:

    bash
    edge-tts --help

    If you see the help output, the installation was successful.

    • Using Your Virtual Environment Later

    Whenever you want to use edge-tts in the future:

    • 1. Navigate to your project directory: bash cd ~/my-edge-tts-project
    • 2. Activate the virtual environment:
      bash
      source venv/bin/activate
    • 3. Use edge-tts as normal
    • 4. When done, deactivate the environment:
      bash
      deactivate

      • Quick Reference

      Task Command
      Create venv python3 -m venv venv
      Activate source venv/bin/activate
      Install package pip install package-name
      Deactivate deactivate

      Troubleshooting...

      "python3 -m venv: command not found" -- Install the venv module:
      bash
      sudo apt install python3-venv

      "python3 not found" -- Install Python 3:
      bash
      sudo apt install python3

      Still getting "externally-managed-environment" error -- Make sure the virtual environment is activated (you should see(venv) in your prompt). If not, run source venv/bin/activate.

      What Gets Installed in the venv Folder?

      The venv folder contains a complete isolated Python installation. You can safely delete the entire venv folder to remove all installed packages and the virtual environment. Your project files remain separate.

      Converting HTML Files to Speech

      Once edge-tts is installed, use the html_to_speech.py script to convert HTML files to audio:

      Setup the Script

      • 1. Save the HTML-to-speech Python script as html_to_speech.py in your project directory
      • 2. Make it executable: bash
        chmod +x html_to_speech.py

      Usage Examples

      Convert HTML to MP3 file:

      bash
      python3 html_to_speech.py mypage.html output.mp3

      Convert with a specific voice:

      bash
      python3 html_to_speech.py mypage.html output.mp3 "en-US-AriaNeural"

      Stream to speakers (no file saved):

      bash
      python3 html_to_speech.py mypage.html

      Finding Available Voices

      List all available voices:

      bash
      edge-tts --list-voices

      Common voices include:

      • en-US-EmmaMultilingualNeural (default)
      • en-US-AriaNeural
      • en-US-GuyNeural
      • en-GB-SoniaNeural
      • en-AU-NatashaNeural
